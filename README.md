# Scholarship Management System (SMS)

**Developer:** Marcjan Villariasa

---

# Project Description
The Scholarship Management System (SMS) is a web-based application designed to help manage scholarship programs, student applications, and administrative evaluation processes.  

The system allows students to apply for scholarships online while enabling administrators to review applications, verify documents, rank applicants, and generate reports for scholarship distribution.

---

# Feature List

## 1. User Authentication & Profile Management

### User Registration
- Register as Student Applicant
- Register as Scholarship Provider / Admin

### Secure Access
- Login
- Logout
- Reset Forgotten Password

### Profile Handling
- View Profile Details
- Edit Personal and Academic Information

---

## 2. Scholarship Program Management

### Program Catalog
- Create / Add New Scholarship Program
- Edit Program Criteria and Deadlines
- View List of Available Scholarships
- Delete or Archive Expired Programs

---

## 3. Application Submission & Tracking

### Document Management
- Upload Required Documents
- Delete or Replace Uploaded Files

### Application Processing
- Submit Final Application
- View Application Status
  - Pending
  - Approved
  - Rejected

---

## 4. Administrative Evaluation & Selection

### Applicant Screening
- Display All Applicant Data
- Verify Submitted Documents
- Rank Applicants based on GWA and Family Income

### Decision Making
- Approve Scholarship Application
- Reject Application with Remarks

---

## 5. Disbursement & Reporting

### Financial Tracking
- Record Scholarship Grant Disbursement Date
- View Payment History of Scholars

### System Reports
- Generate List of Active Scholars (PDF / Excel)
- Generate Scholarship Budget Utilization Report

---

# System Technology

The system is developed using the following technologies:

- Laravel 10 Framework
- PHP 8.1+
- MySQL Database
- Composer Dependency Manager
- XAMPP Local Server
- Postman for API Testing
- Visual Studio Code as the Code Editor
